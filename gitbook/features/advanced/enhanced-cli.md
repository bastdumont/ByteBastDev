# Enhanced CLI

A rich experience built on top of the core CLI, providing better UX and visibility.

## Highlights

- Interactive prompts and summaries
- Colorized logs and progress bars
- Dry-run planning and diff previews

## Common Commands

```bash
# Interactive mode
python orchestrator/main.py

# Execute single task
python orchestrator/main.py --task "Create React app"

# Show plan only
python orchestrator/main.py --task "..." --dry-run
```
