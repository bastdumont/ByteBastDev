# Quick Examples

Copy-paste friendly commands to get started fast.

## React To-Do App

```bash
python orchestrator/main.py --task "Create a React to-do app with localStorage" --output-dir ./output/todo
```

## FastAPI Backend

```bash
python orchestrator/main.py --task "Create FastAPI backend with /health" --output-dir ./output/fastapi
```

## Next.js SaaS Starter

```bash
python orchestrator/main.py --task "Create Next.js SaaS starter" --output-dir ./output/next-saas
```
