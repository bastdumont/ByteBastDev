# CLI Commands

## Main

```bash
python orchestrator/main.py [--task "..."] [--dry-run] [--include-tests] [--output-dir PATH]
```

## Options

- `--task` describe what to build
- `--dry-run` preview plan only
- `--include-tests` generate tests
- `--output-dir` choose output path
- `--log-level` DEBUG|INFO|WARN|ERROR
