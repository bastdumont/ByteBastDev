# Vue SPA Template

Single Page Application with Vue and routing.

## Tech Stack

- Vue 3
- Vue Router
- Pinia (optional)

## Features

- Page routing and layout
- Example components and state

## Getting Started

```bash
python orchestrator/main.py --task "Create Vue SPA" --output-dir ./output/vue-spa
cd output/vue-spa
npm install && npm run dev
```
