# React Dashboard Template

Clean dashboard with table, filters, and charts.

## Tech Stack

- React 18
- Vite or CRA (depending on generation)
- Chart library (e.g., Chart.js)

## Features

- Responsive layout
- Data table with filtering
- Chart component and service layer

## Getting Started

```bash
python orchestrator/main.py --task "Create React dashboard" --output-dir ./output/react-dashboard
cd output/react-dashboard
npm install && npm run dev
```
