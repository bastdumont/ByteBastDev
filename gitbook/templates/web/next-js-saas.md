# Next.js SaaS Template

Production-ready SaaS starter with authentication and billing hooks.

## Tech Stack

- Next.js 14, React 18
- Tailwind CSS
- (Optional) Stripe for subscriptions

## Features

- Auth-ready pages (login/register)
- Dashboard layout with navigation
- Pricing page and subscription flow (when enabled)

## Getting Started

```bash
python orchestrator/main.py --task "Create Next.js SaaS starter" --output-dir ./output/next-saas
cd output/next-saas
npm install && npm run dev
```
