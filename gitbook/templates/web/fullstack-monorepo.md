# Full-Stack Monorepo Template

Frontend + backend managed in a single repository.

## Structure

```
fullstack/
├── frontend/
└── backend/
```

## Features

- Shared tooling and scripts
- Consistent linting and formatting
- Sample CI suggestions

## Getting Started

```bash
python orchestrator/main.py --task "Create fullstack monorepo" --output-dir ./output/fullstack
```
